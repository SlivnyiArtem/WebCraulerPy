Авторы: Сливный Артем, Агафонов Дмитрий

Описание: Многопоточный Web - Crauler, сохраняющий html - страницы на диск c возможностью докачки.

Параметры запуска:

--t [tm или or] - в сучае tm программа выполныет загрузку страниц многопоточно без возможности докачки,
                  в случае or программавыполныет загрузку страниц однопоточно с возможностью докачки,

--domains [] - список доменов по которым может переходить программа(1 или более)
--page [] - url - адрес начальной страницы
Опционально:
--offset [] - смещение для докачки
--updateFiles 1 - обновляет сохраненные html - файлы

Пример запуска: python main.py -t tm --domains ru.wikipedia.org --page https://ru.wikipedia.org/wiki/%D0%9F%D0%B0%D0%B9%D0%BD%D0%B5_(%D1%80%D0%B0%D0%B9%D0%BE%D0%BD) --updateFiles 1
